GreatUni Hackathon

During the GreatUni Hackathon, I collaborated with a team to develop a winning full-stack prototype for Capital One. The project involved:

  • Designing a user-friendly interface using HTML, CSS, and React.js
  • Building a robust back-end system with Python and REST APIs
  • Establishing a database to manage user transactions efficiently
  • Showcasing the prototype with interactive demos to judges

This experience enhanced my skills in full-stack development and teamwork under tight deadlines.

About the Project

The GreatUni Hackathon project involved developing "Kredia," a modern web application designed to streamline user financial interactions. This full-stack project was developed for Capital One during GUH 2023, showcasing innovation, teamwork, and technical expertise in web development.
